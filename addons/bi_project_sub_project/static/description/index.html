Project Sub Project Management Odoo Apps

Project Planning with Sub-projects Odoo Apps

Sub Project Management Odoo App helps users to create sub projects from parent projects. Administrators can set project co-ordinator rules for other users which helps to show only his/her own projects and sub projects. When a user deletes a project which has sub projects and deletes sub projects which have tasks it will display a warning.

Features

Set Project Co-Ordinator Rule for User

You can set project co-ordinator rule for user that shows only his projects and sub-projects.

Easy to Create Sub Projects from Parent Project

You can easily create sub projects from parent project in 'Sub Projects' tab.

Easily Divide Projects and Sub Projects

You can easily divide project and sub projects with its details.

Display Warning on Deletion of Project and Sub Project

When you delete project which has sub projects and delete sub project which has tasks it will display warning.

Set Project Operation for User

Under user configuration you can set project operation as Project Co-Ordinator.

Add Sub Projects

Goto Project -> Configuration -> Projects under 'Sub Projects' tab you can add sub projects of parent project.

Added Sub Projects

You can see sub projects are added and click on 'Sub Projects' smart button to open sub projects.

Display Sub Projects List

Display All Projects List with Different Project Manager

You can see list of all projects with different project manager in list view.

Display Own and Sub Projects

You can see 'Project Co-Ordinator' access group user has show his own projects and sub projects.

You can see another 'Project Co-Ordinator' access group user has show his own projects and sub projects.

Delete Project Which has Sub Projects

You can not delete project which has sub projects otherwise it will be display warning.

Delete Sub Project Which has Tasks

You can not delete sub project which has tasks otherwise it will be display warning.
