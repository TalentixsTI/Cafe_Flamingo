Delete POS Order


The "Delete POS Order" Module Streamlines The Removal Of Paid Or Posted Point Of Sale Orders Users Can Choose Between Two Configurable Options"Delete POS Order Without Code" And "Delete POS Order With Code"Providing Flexibility For Deleting Single Or Multiple Orders According To Their Requirements Within The Odoo Platform.

Key Features

User Can Delete Multiple POS Orders
Users Can Choose The Delete POS Order Option
Free Support For 90 Days

Enable 'POS Delete Order Access' In The "Settings" Under The "Users" Section.


In The Point Of Sale , Under Configuration Settings, Users Can Choose The "Delete POS Order" Option Using A Radio Button.


Select The "Delete POS Order" Button In The POS Order Form View To Delete A Paid Order.


Users Can Delete Multiple Paid Order From The List View.


Upon Clicking The "Delete POS Order" Button, A Confirmation Pop-up Will Appear To Confirm The Order Deletion.


Go To POS Settings, Select 'Delete POS Order With Code' And Type In A Code To Make It Work.


Users Also Have The Capability To Delete Invoiced Orders.


Additionally, Users Can Delete Multiple Invoiced Orders Directly From The List View.


When You Click "Delete POS Order" A Pop-up Will Show Up, And You'll Need To Enter A Code To Confirm The Order Deletion.


If The User Inputs An Incorrect Security Code, They Will Receive A Validation Message Stating That The Entered Code Is Invalid.

What are the benefits of using the "Delete POS Order" module in Odoo?

  • It simplifies the process of removing paid or posted point of sale orders, providing flexibility and efficiency to users.

What happens if I enter the wrong security code during the deletion process?

  • Entering an incorrect security code will prompt a validation message, notifying that the code is not valid for the deletion of POS orders within the module functionality.

Can I customize the security code required for deleting POS orders

  • Yes, users have the flexibility to configure their security preferences, choosing between 'Delete POS Order Without Code' or 'Delete POS Order With Code' in the module settings.
